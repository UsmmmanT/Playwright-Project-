# Playwright-Project-
A clean Playwright starter template with pre-configured test structure, helpers, and GitHub Actions integration for fast test setup.

## Startup
1) Install dependencies:
	- `npm install`
2) Run all tests:
	- `npx playwright test`
3) Run a specific spec file:
	- `npx playwright test playwright-api-framework/tests/crud/posts.crud.spec.js`
4) Open the HTML report after a run:
	- `npx playwright show-report`
